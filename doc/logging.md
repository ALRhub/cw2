**cw2** comes with a a variety of logging capabilities. This document will explain how to use the basic "Console" logging to document `print()` statement. An Interface class is provided with which you can define your own custom Logging class. Additionally, **cw2** provides advanced logging functionality in form of a [Pandas Dataframe](https://pandas.pydata.org/) Logger, for Excel-like table structures, and a [Weights & Biases (WandB)](https://wandb.ai/site) Logger for advanced metrics.
